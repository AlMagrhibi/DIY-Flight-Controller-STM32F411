# STM32F411CEU6 Pinout

| Function | STM32 Pin |
|----------|------------|
| MPU6500 SCL | PB8 |
| MPU6500 SDA | PB9 |
| GPS TX | PA10 |
| GPS RX | PA9 |
| LoRa TX | PA2 |
| LoRa RX | PA3 |
| Motor 1 PWM | PA6 |
| Motor 2 PWM | PA7 |
| Motor 3 PWM | PB0 |
| Motor 4 PWM | PB1 |
| Status LED | PC13 |

---

# Timers

| Timer | Usage |
|--------|--------|
| TIM3 CH1 | Motor 1 |
| TIM3 CH2 | Motor 2 |
| TIM3 CH3 | Motor 3 |
| TIM3 CH4 | Motor 4 |
