# DIY Flight Controller STM32F411

Custom DIY Flight Controller based on STM32F411CEU6 Black Pill.

## Hardware

- STM32F411CEU6 Black Pill
- MPU6500 IMU
- NEO-6M GPS
- LoRa AS32 TTL-1W
- PS5 Controller + ESP32 Ground Station

## Features (Planned)

- Custom Flight Stack
- Complementary Filter
- PID Stabilization
- GPS Position Hold
- LoRa Telemetry
- Failsafe System
- Return To Home (Future)
